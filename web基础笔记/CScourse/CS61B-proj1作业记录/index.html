Packages

当想要声明当前code是哪个包时，需要在文档开头使用package关键字。

当想要使用一个包中的一个类或者方法时，可以通过写它的全名，e.g. deque.ArrayDeque，也可以使用import关键字导入这个类或者包。

这样在之后使用的时候就可以使用简写ArrayDeque。

The Deque API

大致就是让我们了解，接下来需要做出来两个类ArrayDeque和LinkedListDeque，一个是基于数组实现，另一个则是基于链表，它们都需要实现Deque接口。
