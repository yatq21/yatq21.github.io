堆叠注入写shell

刚开始fuzz，发现%5c有不一样的回显，但不明白只是什么东西，直到看到其他师傅wp才反应过来是注释符发力了（

想到常见的后台查询语句：$sql = "SELECT * FROM users WHERE username = '$username' AND password = '$password'";
